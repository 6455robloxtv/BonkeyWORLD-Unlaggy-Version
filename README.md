# bonziworld.com-edited
Just an random BonziWORLD server.
This should be for bonziworld.com users i hope.
## Dependencies
- Node.js and npm
- Ruby
- Sass
- Git
- Cordova (Optional)

## Setup
In a terminal/command prompt, navigate to where you'd like BonziWORLD to be placed and run the following:
```
git clone https://github.com/ItzUltraChace/bonziworld.com-edited
cd bonziworld.com-edited
```

### Client
```
cd src
npm install
grunt build_www
cd ..
```

### Server
```
cd server
npm install
node index.js
```
After this, BonziWORLD will be accessible on port 80. (http://localhost/)
